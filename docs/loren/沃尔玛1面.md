## 岗位JD  
财务结算  
岗位职责  
1. 负责财务结算（账单，发票，付款）业务财务后端开发工作  
2. 负责系统模块开发与设计，输出接口文档及技术方案文档  
3. 按时按质完成模块开发任务，充分自测，完成与前端的联调对接  
4. 发现和解决业务系统的技术问题，保证系统的性能和稳定性  
5. 理解公司业务，能基于业界先进的技术对系统和业务流程进行优化  
任职资格  
1. 计算机相关专业本科及以上学历，五年以上JAVA开发工作经验；   
2. 扎实的java编程基础，熟悉多线程，熟悉常用的Java开源框架, 比如Spring、SpringMVC、MyBatis、SpringBoot；   
3. 熟悉常见的中间件、了解其原理：分布式缓存、消息中间件，负载均衡、连接池等，比如Nginx、Kafka、Redis cluster;  熟悉Spring Cloud,  Nacos, Consul；熟练MySQL，Oracle 、 MS SQL Server；   
4. 熟悉DDD建模思想，具有一定的业务架构设计能力、业务建模经验者优先；善于捕捉业务需求、架构设计中存在的问题，并给出有效的解决措施和方案；  
5.  热爱技术，工作认真、严谨，具备较强责任心，具有一定的抗压能力，能自我激励自我驱动，善于沟通与团队协作。
